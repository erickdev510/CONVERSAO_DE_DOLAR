# CONVERSOR DE MOEDAS 💵
O projeto de Conversor de Moedas foi desenvolvido em Python com o objetivo de criar uma aplicação simples capaz de realizar a conversão de valores entre Dólar Americano (USD) e Real Brasileiro (BRL). O sistema permite ao usuário informar um valor e escolher o tipo de conversão desejada, facilitando cálculos de câmbio de forma rápida e prática. Durante o desenvolvimento, foram utilizados conceitos fundamentais da programação, como variáveis, operadores matemáticos, estruturas condicionais, funções e entrada de dados pelo terminal, proporcionando experiência prática na criação de aplicações funcionais em Python.

## Sobre o projeto ⚙️
O conversor foi criado para auxiliar usuários que precisam transformar valores entre as moedas dólar e real de maneira simples e eficiente. O programa recebe o valor informado pelo usuário, aplica a cotação definida no sistema e exibe o resultado da conversão de forma organizada. Esse projeto contribuiu para o aprendizado sobre cálculos matemáticos, manipulação de dados e desenvolvimento de menus interativos, além de fortalecer a lógica de programação e a organização do código.

## Conclusão 🚀
O desenvolvimento do projeto de Conversor de Moedas foi importante para aprimorar conhecimentos em Python e compreender melhor a criação de sistemas baseados em cálculos e interação com o usuário.
